A personal utility · runs in your browser

Valhallist

Paste a list of songs, one per line. It gathers each one from Spotify, shows you what it found, and lets you fix anything uncertain before the playlist is assembled.

First time? Connect your Spotify app (one-time, ~5 min)
  1. Go to the Spotify Developer Dashboard and click Create app.
  2. Name it anything (e.g. "My Playlist Tool"). For Redirect URI, add the exact address this page is open at — shown below — then save.
  3. In the app's settings, copy the Client ID and paste it below.
  4. Under APIs used, make sure Web API is checked.

Stored only in this browser (localStorage), never sent anywhere but Spotify.

Step 1 — Your list

One song per line. Artist - Title works best, but Title - Artist and plain Title are handled too. Blank lines and # comments are ignored.

Step 2 — Review the matches

Confident matches are kept automatically. Flagged rows found a result but weren't sure — check them, swap to an alternative, or drop them. Then create the playlist.
